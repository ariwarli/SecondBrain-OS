# Obsidian Structure: PARA + Authority Framework

Struktur folder ini dirancang untuk kecepatan akses dan pemisahan yang jelas antara informasi aktif, riset, dan produksi konten.

---

## 📂 Struktur Utama

### 00 Inbox
- **Fungsi:** Tempat pendaratan capture dari Telegram (Text/Audio).
- **Aturan:** Folder ini harus kosong di akhir minggu. Semua file harus dipindahkan ke folder 10, 20, 30, atau 40.

### 10 Projects
- **Fungsi:** Catatan untuk hal yang memiliki *Deadline* dan *Output* spesifik.
- **Contoh:** `/Launching SBF Kit`, `/Client Project Alpha`, `/Rebranding Business`.
- **Aturan:** Setiap proyek memiliki sub-folder sendiri.

### 20 Areas
- **Fungsi:** Tanggung jawab jangka panjang yang menuntut standar performa tinggi.
- **Contoh:** `/Health`, `/Finances`, `/Legal`, `/Marketing Strategy`.

### 30 Resources
- **Fungsi:** Topik minat berkelanjutan atau riset statis.
- **Contoh:** `/AI Tools Research`, `/Economics`, `/Philosophy`, `/Industry Reports`.
- **Aturan:** Ini adalah "Perpustakaan" otak kedua Anda.

### 40 Archives
- **Fungsi:** Proyek yang sudah selesai atau area yang tidak lagi aktif.
- **Contoh:** `/Client Project 2025`, `/Old Website Assets`.

---

## 📂 Folder Khusus: 50 Authority Architecture
Ini adalah folder rahasia untuk membangun otoritas Anda sebagai ahli.

### 51 Deep Insights
- Berisi 1 wawasan tajam hasil gabungan dari catatan di folder Resources.
- AI dapat membantu merangkum ini menggunakan *Synthesis Prompt*.

### 52 Content Drafts
- Draft mentah untuk Newsletter, LinkedIn, atau Twitter Threads.

### 53 Strategic Conversations
- Catatan dari percakapan berharga yang muncul setelah Anda memposting konten.
- Ini adalah sumber *Lead* terbaik Anda.

---

## 📂 Folder Penunjang (Auto-generated)
- `_Assets/`: Tempat menyimpan gambar, PDF, dan rekaman audio asli.
- `_Templates/`: Berisi template file untuk meeting note, project plan, dll.

---

## Tips Organisasi
1. **Don't Over-organize:** Jangan buat terlalu banyak sub-folder di awal.
2. **Naming Convention:** Gunakan tanggal di depan nama file penting (Contoh: `2026-04-17 - Meeting Client X`).
3. **Internal Links:** Gunakan `[[Nama File]]` untuk menghubungkan satu ide ke ide lainnya. Inilah kekuatan asli Obsidian.
