## Writing Style Blog

### Karakteristik Tulisan Bani Risset

1. **Gaya bahasa:** Santai tapi berbobot, gw-lo register untuk konten sosmed, formal untuk artikel blog
2. **Tone:** Praktis, langsung ke poin, tidak bertele-tele
3. **Format:** 
   - Step-by-step dengan screenshot
   - Listicle dengan kategori jelas
   - FAQ di akhir artikel
4. **Panjang:** 1500-3000 kata untuk artikel reguler, 5000+ untuk pillar

### Struktur Artikel Ideal

```
[H1] Judul dengan keyword + tahun
[Intro] Hook + promise + outline (3-4 kalimat)
[H2] Apa itu [Topic]
[H2] Kenapa penting untuk [Target]
[H2] Cara [Action] — Step-by-step
  [H3] Step 1...
  [H3] Step 2...
[H2] Tools yang direkomendasikan
[H2] FAQ (5 pertanyaan — schema markup)
[CTA] Link ke halaman jasa atau artikel terkait
```

### Content Format Guidelines

| Tipe Konten | Panjang | Struktur |
|-------------|---------|----------|
| How-To Tutorial | 1500-2000 kata | Steps + screenshots + FAQ |
| Comparison | 2000-3000 kata | Tabel + pros/cons + rekomendasi |
| Curated Listicle | 2500-3500 kata | 15-25 items dengan kategori |
| Pillar/Mega Guide | 4000-6000 kata | Multi-section + TOC + downloadable |
| Artikel Update | +500-1000 kata | Add FAQ + refresh data + internal link |
