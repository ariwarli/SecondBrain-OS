# Status REED / Hermes — 2026-04-30

## ✅ Yang Sudah Diselesaikan

### 1. Telegram Slash Menu Fix
**Masalah:** Tombol slash menampilkan semua command (`/branch`, `/compress`, `/rollback`, etc) padahal seharusnya hanya `/menu`.
**Penyebab:** Telegram client cache command list lama.
**Fix:** Menambahkan `delete_my_commands()` sebelum `set_my_commands()` di code.
**Status:** Code deployed ke VPS, menunggu restart Telegram client untuk clear cache.

### 2. `/menu` “Menu closed”
**Ini bukan bug.** Saat user membuka inline launcher lalu menekan tombol “Close”, launcher message di-edit jadi “Menu closed”.
**Solusi:** Jika ingin membuka kembali, ketik `/menu` lagi. Saran: jangan tekan “Close” kecuali memang ingin menutup launcher.

### 3. Bot “Bolot” (Lambat)
**Root cause:** `model.default = qwen3-coder:480b` → model 480B parameter, sangat lambat.
**Fix:** Ganti ke `deepseek-v4-flash` (model lebih kecil, lebih cepat) di `/home/hermes/.hermes/config.yaml`.
**Status:** Config sudah di-update, gateway sudah di-restart dengan model baru.

## 🔧 Model Selection — Saat Ini vs Otomatis

### Apa yang Saya Temukan

Hermes saat ini **tidak punya** model selection otomatis berdasarkan context/topic/channel. Selection model bekerja seperti ini:

```
Incoming Message
    |
    v
[Gateway] _resolve_session_model_and_runtime()
    |
    +--> config.yaml → model.default (sekarang: deepseek-v4-flash)
    |
    +--> per-session override? (/model command)
    |
    +--> fallback_providers (kalau primary fail)
    |
    v
 AIAgent menggunakan model yang terpilih
```

### Model Aliases yang Sudah Tersedia

Config sudah punya model aliases yang bisa dipilih manual via `/model`:

| Alias | Model | Provider | Gunakan Untuk |
|---|---|---|---|
| `speedup-dev` | qwen3-coder:480b | ollama-cloud | Coding kompleks |
| `speedup-strategic` | kimi-k2.6 | ollama-cloud | Strategic thinking |
| `speedup-ops` | gpt-oss:120b | ollama-cloud | Operations |
| `speedup-fast` | deepseek-v4-flash | ollama-cloud | ⚡ Cepat, default baru |
| `speedup-brand` | gemma4:31b | ollama-cloud | Brand content |
| `speedup-research` | glm-5.1 | ollama-cloud | Research |

### Cara Memilih Model Manual

Di Telegram, ketik:
```
/model speedup-fast
```

Atau gunakan `/menu` → pilih “Model” untuk browse model picker.

## 🚧 Yang Perlu Dilakukan Selanjutnya

### 1. Otomatis Model Selection Berdasarkan Context
Hermes belum punya fitur ini. Untuk otomatisasi, perlu:
- Modifikasi code gateway: intercept pesan sebelum dikirim ke AIAgent
- Parse topic/channel dari SessionSource
- Lookup model_alias dari config.yaml berdasarkan topic
- Ganti model di runtime sebelum invoke agent

Ini adalah **feature request** yang perlu di-develop, bukan quick fix.

### 2. Model Selection Sementara (Workaround)
Gunakan **per-session `/model` command** — tiap kali mulai kerja di task tertentu, ketik `/model <alias>` sebelum mulai ngobrol.

### 3. Restart Telegram Client
Untuk clear cached command list lama dan verify slash menu hanya menampilkan `/menu`.
