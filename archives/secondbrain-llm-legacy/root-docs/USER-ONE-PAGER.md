# USER ONE PAGER

Kalau Anda cuma mau ingat hal paling penting, baca file ini.

## Kirim ke mana?

- Tidak yakin? `Inbox`
- Sudah jelas task? `Tasks`
- Orang / follow-up? `Personal CRM`
- Konten? `Content`
- Riset / knowledge? `Knowledge Base`
- Debug / VPS / config? `Ops`
- Personal / capek / overwhelm? `Wellbeing`

## Cara mulai kerja

Copas ini:

```text
Gw mulai kerja sekarang. Buka konteks aktif untuk lane ini, ringkas status sekarang, lalu kasih:
1. keputusan aktif
2. blocker
3. next action paling penting
4. apa yang berubah terakhir kalau ada
```

## Cara update

Copas ini:

```text
Update konteks:
[tulis update di sini]

Kalau ini mengubah keputusan, blocker, atau next action aktif, update memory aktifnya juga.
Kalau tidak, cukup simpan sebagai histori yang relevan.
```

## Cara simpan sesi penting

Copas ini:

```text
Save upload session ini ke knowledge-base.

Ringkas hanya:
- keputusan
- fakta baru
- blocker
- next action
```
