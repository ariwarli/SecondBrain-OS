Ide 4: "CRM Klien Hukum" — Track Relasi dan Follow-up via Suara
Problemnya:
Pengacara sering lupa follow-up, atau tidak tahu status percakapan terakhir dengan klien karena tercampur di WhatsApp.
Yang lo tawarkan:
Setiap selesai ngobrol dengan klien, rekam voice note singkat:

"Baru telepon Pak Andi soal kasus warisan. Dia minta gw kirim draft surat kuasa minggu ini. Dia agak nervous soal timeline."

Sistem otomatis:

Update status klien di CRM
Set reminder "kirim draft surat kuasa — deadline minggu ini"
Catat konteks emosional (nervous soal timeline → handle dengan hati-hati)
