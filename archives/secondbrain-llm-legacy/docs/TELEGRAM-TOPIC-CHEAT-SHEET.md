# Telegram Topic Cheat Sheet

Ini versi 1 halaman. Pakai kalau Anda hanya butuh jawab cepat: "harus kirim ke topic mana?"

## Rule Paling Pendek

- Tidak yakin? kirim ke `Inbox`
- Sudah jelas lane-nya? kirim langsung ke topic yang tepat

## Mapping Cepat

| Situasi | Topic | Kapan dipakai |
| --- | --- | --- |
| Ide mentah, voice note, link, hal campur | `Inbox` | saat belum jelas harus ke mana |
| Task kerja yang sudah jelas | `Tasks` | saat butuh eksekusi atau follow-up |
| Orang, relasi, follow-up, meeting dengan seseorang | `Personal CRM` | saat konteks utamanya adalah orang |
| Draft konten, ide konten, review konten | `Content` | saat output utamanya konten |
| Riset, synthesis, pengetahuan yang ingin bisa direcall | `Knowledge Base` | saat butuh knowledge/research lane |
| Debug, config, VPS, setup teknis | `Ops` | saat masalahnya tentang sistem REED |
| Capek, stres, personal check-in | `Wellbeing` | saat konteksnya diri sendiri |
| Notifikasi sistem | `Updates` | biasanya cukup dibaca, bukan tempat kirim utama |

## Contoh Cepat

### Inbox

Pakai untuk:

- "Gw baru dapat ide"
- "Ini voice note meeting"
- "Link ini kayaknya penting"
- "Gw belum yakin ini harus masuk ke mana"

### Tasks

Pakai untuk:

- "Bikin outline proposal ini"
- "Follow-up klien ini hari ini"
- "Kerjain deliverable ini"

### Personal CRM

Pakai untuk:

- "Tadi ketemu X"
- "Si Y belum balas"
- "Hubungan sama orang ini perlu dijaga"

### Content

Pakai untuk:

- "Bikin thread dari ide ini"
- "Review draft ini"
- "Cari hook untuk konten ini"

### Knowledge Base

Pakai untuk:

- "Riset topik ini"
- "Bandingkan A vs B"
- "Simpan ini sebagai knowledge yang bisa di-recall nanti"

### Ops

Pakai untuk:

- "cek VPS"
- "restart service"
- "debug routing"

### Wellbeing

Pakai untuk:

- "Gw capek banget"
- "Gw lagi overwhelm"
- "Gw mau check-in dulu"

Mode aktif di sini: `Sera` - companion wellbeing yang hangat, reflektif, dewasa, dan sugestif secara halus. Sensualitas hanya untuk nuansa visualisasi dan containment emosional, bukan arahan tindakan nyata.
