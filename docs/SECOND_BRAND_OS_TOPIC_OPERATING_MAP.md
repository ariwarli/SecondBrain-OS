# Second Brand OS Topic Operating Map

## `Command Center`
- tujuan: keputusan besar, review lintas lane, brainstorming makro
- input: strategi, tradeoff, review banyak lane
- output: keputusan + handoff
- jangan dipakai untuk output final lane-spesifik yang panjang

## `INBOX`
- tujuan: intake mentah dan routing
- input: semua hal yang belum jelas lane-nya
- output: ack singkat + route
- jangan dipakai untuk kerja aktif

## `tasks`
- tujuan: aksi nyata, eksekusi, audit, revisi
- input: task, next action, implementasi
- output: action list, progress, completion

## `content`
- tujuan: draft output publik
- input: ide, framework, brief, angle
- output: post, thread, carousel, script

## `personal-crm`
- tujuan: relasi dan follow-up
- input: orang, kontak, meeting, deal
- output: context note, next action, reminder

## `knowledge-base`
- tujuan: source, ringkasan, synthesis, candidate canon
- input: URL, PDF, referensi, insight luar
- output: summary, concept, framework candidate

## `ops`
- tujuan: runtime, infra, debug, automation
- input: error, issue, health check
- output: diagnosis, fix, incident lesson

## `updates`
- tujuan: laporan dan alert
- input: progress, status, ringkasan, incident singkat
- output: concise update

## `Hormozi`
- tujuan: offer, pricing, funnel, value equation, sales copy
- input: produk, jasa, source bisnis, angle acquisition
- output: offer draft, pricing logic, sales angle
