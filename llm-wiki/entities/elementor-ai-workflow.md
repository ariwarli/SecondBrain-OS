# Elementor + AI Landing Page Workflow

**Workflow:** Membuat landing page profesional dalam ~5 menit tanpa coding dengan kombinasi **Elementor (WordPress)** + **Claude AI**.

## How It Works

1. Prompt Claude AI untuk generate struktur landing page (HTML + JSON format)
2. Download file JSON Elementor template yang dihasilkan AI
3. Import JSON ke Elementor di WordPress
4. Publikasikan & sesuaikan manual

## Key Tools

- **Elementor** — WordPress page builder drag-drop, fitur **Container** (terbaru, ganti Section)
- **Claude AI** (sonet) — untuk generate HTML + JSON Elementor template
- **WordPress** — CMS platform

## Why Claude > ChatGPT/Gemini for This

Menurut pembuat video, Claude lebih unggul untuk generate kode (HTML, JSON) dibanding ChatGPT atau Gemini.

## Pricing

- Claude AI versi **gratis** cukup untuk demo sederhana
- Untuk proyek kompleks perlu upgrade ke versi **berbayar**

## Why Relevant for Bani

Workflow ini relevan buat klien yang butuh landing page cepat — bisa dipake sebagai value-add di paket consulting atau dijadiin produk digital (template + tutorial).
