# AIMurah

Platform AI API dengan akses 27+ model premium (Claude Sonnet 4.5, GPT-5, Gemini 3.1, dll) via satu API key. Drop-in replacement OpenAI-compatible.

## Keunggulan
- **1 Billion Token gratis** untuk user baru (free tier: 100 req/hari)
- No credit card required untuk free tier
- Support semua tool OpenAI-compatible: Cursor, VS Code, Cline, Continue, Windsurf, OpenCode
- Pricing transparan dalam IDR (sudah termasuk PPN)

## Model Highlights
- **Free**: Sonnet 4.5, Haiku 4.5, DeepSeek 3.2, MiniMax M2.x
- **Pro**: Opus 4.6, GPT-5.4, Gemini 2.5/3, Kimi K2.5, o3/o4 Mini
- **Daily packs** tersedia untuk usage spike (Rp 29k–99k/hari, unlimited req)

## Relevansi untuk Bani

- Testing model performance sebelum commit ke provider utama
- Daily pack cocok untuk campaign burst atau prototyping cepat
