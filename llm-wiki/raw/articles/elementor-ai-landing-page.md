## Step-by-Step Workflow

### 1. Persiapan Awal
- Daftar/Login ke Claude AI (link di deskripsi video) – versi gratis cukup untuk demo sederhana.
- Install WordPress dan Elementor (plugin).

### 2. Membuat Halaman Baru di WordPress
- Buka Pages → Add New → beri judul (misal: "Testing").
- Publish halaman tersebut.
- Klik Edit with Elementor.

### 3. Setting Halaman di Elementor
- Pengaturan → Hidden Title (centang).
- Page Layout → Canvas (agar full-width, cocok untuk landing page).

### 4. Generate Konten dengan Claude AI
- Masukkan prompt ke Claude AI.
- Tunggu hingga selesai (bisa beberapa menit, tergantung kompleksitas).
- Download file JSON yang dihasilkan.

### 5. Import Template ke Elementor
- Di editor Elementor, klik ikon folder (Template Library).
- Pilih Save Template → Import Template.
- Upload file JSON → Import → Continue.
- Setelah terimport, klik Insert pada template yang muncul.

### 6. Publikasikan & Sesuaikan
- Klik Publish.
- Preview halaman.
- Lakukan penyesuaian manual: font, teks, padding, margin, button, dll.

## Important Notes

| Tools | Elementor (WordPress) + Claude AI (bisa kombinasi AI lain) |
| Format Output | HTML → JSON Elementor (container-based) |
| Biaya | Claude AI gratis (terbatas token/kredit); untuk proyek kompleks perlu upgrade |

| Kustomisasi | Hasil AI masih perlu disesuaikan (font, warna, teks, layout) |
| Fitur Elementor | Gunakan Container (bukan Section) – fitur terbaru |
